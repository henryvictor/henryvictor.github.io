Tech Cards Generator

L'outil original utilise l'API mise à disposition par Colormind pour obtenir des palettes de couleurs. Ces palettes sont ensuite filtrées au travers d'un algorithme très simple comparant les valeurs de luminance de chaque couleur afin de les comparer pour finalement éliminer les palettes qui ne fourniraient pas un contraste suffisant (la luminance est utilisée comme proxy afin d'éviter un calcul beaucoup plus compliqué faisant appel aux autres - nombreuses - valeurs qui composent une couleur).

La version présentée ici fonctionne sur un set pré-établi de palettes de couleurs (en dur dans le code) afin d'éviter d'envoyer des requêtes inutiles.
